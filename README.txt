----------


This package contains the German language files (and only these files) of a number of non-core Elgg plugins. The list below shows the plugins included. The number in brackets is the plugin version the translation has been made for (it might work with other versions of the plugins, but I can't give you any guarantee...).

You can find the language files in the corresponding subdirectories of this collection. Included in the subdirectories is also a text file with a link to the download page of the plugin on http://community.elgg.org (or other download resources).

The languange files have been translated completely. Unfortunately, some plugins don't include all language strings in their language files. Therefore, a few phrases in the original language will appear on your Elgg site. I'm afraid, without modifying the source code of these plugins it's not possible to offer a 100% complete translation - least of all make a multi-language site possible. I might update some plugins in future - if my time permits.


I'm sure the translation contains also a few spelling mistakes. If you find any errors in translation, please give me feedback!

I'll try to publish updated releases of this collection when newer versions of plugins get released or to correct errors found. I might also include translations of more plugins in future releases.

Dieses Paket enthält die deutschen Sprachdateien (und nur die Sprachdateien) für eine Anzahl von Plugins, die nicht im Standardpaket von Elgg enthalten sind. Die enthaltenen Plugins sind in der Liste unten aufgeführt. Die Nummer in Klammern gibt die Version der Plugins an, für die die Übersetzung durchgeführt wurde (möglicherweise funktionieren die Sprachdateien auch für andere Versionen der Plugins, aber ich kann es nicht garantieren...).

Die Sprachdateien sind in den entsprechenden Unterverzeichnissen in dieser Sammlung zu finden. In den Unterverzeichnissen gibt es auch jeweils eine Textdatei mit einem Link zur Download-Seite der Plugins auf http://community.elgg.org (oder alternativen Download-Seiten).

Die Übersetzung der Sprachdateien ist komplett. Allerdings sind bei einiger Plugins nicht alle Sprach-Strings in den Sprachdateien enthalten. Daher werden einige Textausgaben auf Deiner Elgg-Seite in der Originalsprache erscheinen. Leider ist es nicht möglich, eine 100%ig vollständige Übersetzung anzubieten, ohne die Quelldateien dieser Plugins zu modifizieren - eine mehrsprachige Seite ist erst recht nicht möglich. Falls ich mal Zeit übrig habe, werde ich vielleicht einige dieser Plugins entsprechend anpassen.


Ich bin mir sicher, dass in den Übersetzungen noch einige Fehler enthalten sind. Bitte meldet mir gefundene Fehler, damit ich sie korrigieren kann!

Ich werde versuchen, aktualisierte Versionen dieser Sammlung zu veröffentlichen, wenn neue Versionen der enthaltenen Plugins erscheinen oder um gefundene Fehler zu berichtigen. Vielleicht wird die Sammlung in zukünftigen Veröffentlichungen auch noch mehr Plugins enthalten.


----------


Install instructions / Installationshinweise:

* Just copy the language file de.php to the corresponding language subfolder in the plugin's directory (mod/plugin-name/language/). In case you have selected "German" as language of your account or the site language is set to German, the translations will be visible automatically.

* Kopiere einfach die Sprachdatei de.php in das entsprechende language-Unterverzeichnis im Verzeichnis des Plugins (mod/plugin-Name/language/). Sofern Du entweder "German" als Sprache für Deinen Account eingestellt hast oder die Standardsprache Deiner Seite auf "German" gesetzt ist, sollten die Übersetzungen automatisch sichtbar sein.
